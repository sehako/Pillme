# 💊복약 관리의 새로운 방법, PILLME

<div align="center">

![alt text](img/아이콘.png)

</div>

## 프로젝트 소개

- PILLME는 복용해야 하는 약물을 처방전 또는 처방 봉투를 기반으로 추가하고, 주기적인 알람 설정을 통해서 매일 정해진 시간에 사용자에게 복약 알림을 제공하는 서비스입니다.
- 관리 대상 등록 기능을 통해 보호자가 복약 관리를 대신 할 수 있습니다.
